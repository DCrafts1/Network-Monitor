
# Network Monitor

>

### Required Libraries:
- [Pcap4j](https://www.pcap4j.org/)
- [Numpy](http://www.numpy.org/)
- [Pandas](https://pandas.pydata.org/)
- [Matplotlib](https://matplotlib.org/)

### How To:
- Install libraries
- Install client software on all network devices
- Install server software on central server
